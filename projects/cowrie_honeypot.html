Cowrie Honeypot Data Aquistion

    The data shown below only portrays some of the GeoIP data collected from the log files. Specifically 3/4/2020, 3/6/2020, 3/10/2020, and 3/14/2020. This map was created using Google GeoCharts. The data listed is only limited to 3 columns so dates could not be added. Some locations with multiple circles on top of each other indicate multiple IP addresses with similar GeoIPs. (They kind of look greyish).

About the project:

Grabbed IP data from a honeypot created with Cowrie. The honeypot was hosted in the IT department at Clarkson University and collected data for about 2 months. This IP data was converted into their respective GeoIP locations using Python and mapped on to a Google GeoCharts web page.


Technology used:

Python, JSON
